2FA LOGIN SYSTEM

About This Project

  • Secure login system using Two-Factor Authentication.
  • User first enters username/email + password.
  • Then system generates an OTP for second verification.
  • OTP is valid for 5 minutes.
  • Passwords are stored securely using hashing.
  • Built using Python Flask + SQLite.
