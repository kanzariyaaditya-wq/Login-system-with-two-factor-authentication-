2FA Login System

This is a cyber security project that provides a secure login system using Two-Factor Authentication. It protects user accounts using password verification and OTP verification.

Project Features

  • User Registration
  • Secure Login System
  • Password Hashing
  • OTP Verification
  • 5 Minute OTP Timer
  • Resend OTP Option
  • Secure Dashboard

Technologies Used

Python, Flask, SQLite, HTML, CSS, JavaScript

How It Works

  • User creates an account.
  • User logs in using username/email and password.
  • System generates OTP.
  • User verifies OTP.
  • Dashboard opens after successful verification.
